Пример использования:

Создание заметки:

Введите "1" для создания заметки.
Введите заголовок заметки.
Введите содержимое заметки.
Чтение списка заметок:

Введите "2" для чтения списка заметок.
Редактирование заметки:

Введите "3" для редактирования заметки.
Введите индекс заметки, которую хотите отредактировать.
Введите новое содержимое заметки.
Удаление заметки:

Введите "4" для удаления заметки.
Введите индекс заметки, которую хотите удалить.
Выход:

Введите "5" для выхода из программы.
